# NES Emulator

制作途中のNESエミュレータ

## 使用ライブラリ
SDL2

## 実装済み機能
- NESTEST.nesのテストをすべてパス
- 入力割り込み
- バックグラウンドレンダラ

## 未実装
- 水平、垂直同期
- スプライトレンダラ(バグ修正)

## 実行
`make run ROMPATH=<Your .nes file pass>`
